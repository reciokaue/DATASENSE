O Net Promoter Score (NPS) é uma métrica que fornece uma visão rápida sobre a satisfação do cliente. Para obter dados precisos e acionáveis, é importante estruturar bem o formulário NPS e incluir elementos complementares. Neste artigo, abordamos as melhores práticas para criar formulários NPS.

### Estrutura Básica do Formulário NPS

O formulário NPS começa com uma pergunta simples: "Em uma escala de 0 a 10, o quanto você recomendaria nossa empresa a um amigo ou colega?" Essa pergunta direta facilita a resposta e permite uma comparação fácil dos resultados ao longo do tempo.

### Pergunta de Acompanhamento

Inclua uma segunda pergunta opcional, como "Por que você deu essa nota?" Isso oferece contexto para a nota e ajuda a identificar áreas específicas de melhoria. As respostas qualitativas também podem revelar tendências que a métrica NPS, sozinha, não captura.

<Callout>
  As perguntas abertas complementares são valiosas para entender o que motiva os Promotores e o que precisa ser corrigido para satisfazer os Detratores.
</Callout>

### Timing e Frequência de Aplicação

É importante aplicar o NPS em momentos-chave, como após uma compra ou o término de um serviço. No entanto, evite aplicar o formulário com muita frequência para não gerar fadiga nos clientes. Revisitar o NPS periodicamente é uma boa prática para monitorar a evolução da satisfação do cliente.

### Análise dos Resultados

Analise os resultados do NPS com frequência e compartilhe os insights com a equipe. Um NPS positivo é um bom sinal, mas o feedback dos Detratores é especialmente valioso para ações de melhoria. Usar esses dados para ajustar estratégias pode fortalecer a lealdade e melhorar o relacionamento com os clientes.

A implementação consistente do NPS e a análise detalhada dos dados coletados criam uma base sólida para aprimorar a experiência do cliente e impulsionar o crescimento.
